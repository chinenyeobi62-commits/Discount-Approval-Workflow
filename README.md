# Discount-Approval-Workflow
Automated discount approval workflow using Microsoft Power Platform to streamline sales processes and improve governance
## Overview
This project involved redesigning and automating the discount approval workflow for a sales team. The goal was to reduce approval delays, centralize tracking, and improve visibility and compliance.

---

## Business Problem
The sales team used a manual approval process for discount requests, which caused:

- Delays in deal closures

- Poor visibility into approval stages

- Inconsistent documentation

- Missed follow-ups

- Revenue leakage risks

Approval decisions were difficult to track, and there was no structured system for accountability.

---

## Objective
Redesign the discount approval workflow to create transparency and structured routing, an automated workflow that ensures timely approvals, centralized tracking, and improved compliance.

---

## Solution Approach
- Analyzed the existing discount approval lifecycle  
- Defined multi-level approval thresholds and decision logic  
- Designed a conditional routing workflow  
- Implemented automated notifications and escalations  
- Centralized documentation in Dataverse  
- Provided user guidance for adoption support  

---

## Tools & Technologies
- Power Automate  
- Dataverse  
- SharePoint  
- Microsoft 365  

---

## Results
- 60% reduction in approval cycle time  
- 80% reduction in missed follow-ups  
- Centralized audit-ready documentation  
- Improved pipeline visibility and governance
- Improved process transparency across sales leadership
